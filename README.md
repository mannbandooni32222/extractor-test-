# Free Email & Social Media Extractor (MVP)

## How to Run Locally

1. Make sure Python is installed (3.8+ recommended).
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
4. Open the web browser link that appears to start using the extractor.

## Features
- Extract up to 5 emails per website
- Extract Facebook and Instagram links
- Bulk input (one URL per line)
